## Getting Started

#### Clone The Project
```
$ git clone https://github.com/ERFouX/Symfony-Blog.git
```
Go to the project directory
```
$ cd Symfony-Blog
```
#### Install Dependencies
```
$ composer install
```
#### Running Project

start your apache server or anything
start mysql server
```
$ Symfony start:server
```
### Migrations

```
$ php bin/console doctrine:migrations:migrate
```
